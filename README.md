# Poster Designer Pro AI Agent Skill v2.0

An AI Agent based visual content creation workflow.

## Features

- Planner Agent
- Writer Agent
- Designer Agent
- Reviewer Agent
- Exporter Agent
- Design Token system
- Reusable templates
- HTML/SVG ready workflow

## Supported Use Cases

- Xiaohongshu carousel posts
- WeChat articles visual cards
- LinkedIn marketing graphics
- AI tutorials
- B2B marketing content
- GEO/SEO content visualization

## Installation

Copy the skill into your Codex skills directory:

```bash
~/.agents/skills/poster-designer-pro
```
